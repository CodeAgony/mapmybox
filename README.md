# mapmybox

React app powered by Mapbox API to display 10 restaurants around my home town center in a user-friendly way.

### Chingu instructions
To run it, 
clone the repo to your local machine,
```sh
$ npm install 
$ npm start
``` 
### Tech used: React & Context, Mapbox API, react-mapbox-gl
